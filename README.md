3D-Printable-DP6037
===================

A case made for the Bus Pirate v3.6 (and other BPs that are DP6037 compatible). [Look here](http://open.konspyre.org/blog/2012/12/31/3d-printed-bus-pirate-case/) to see what it looks like when printed.
